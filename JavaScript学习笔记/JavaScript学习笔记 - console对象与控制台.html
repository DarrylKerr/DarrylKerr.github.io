JavaScript学习笔记 - console对象与控制台

发表于 2016-05-18 | 分类于 JavaScript学习笔记 |

console 对象

console对象是 JavaScript 的原生对象，可以输出各种信息到控制台，并且提供很多辅助方法。
console常见的用途有两个。

  • 调试程序
  • 提供一个命令行接口，用来与网页进行互动

console 对象的静态方法

console.log()

console.log方法用于在控制台输出信息，可以接受一个或多个参数，将它们连接起来输出。

如果第一个参数是格式字符串，console.log方法将依次使用后面的参数替换占位符，然后进行输出。

console.log方法支持的占位符有：

  • %s 字符串
  • %d 整数
  • %i 整数
  • %f 浮点数
  • %o 对象的链接
  • %c CSS 格式字符串

console.info(), console.debug()

和console.log类似，可以理解为同一个东西.

console.warn(), console.error()

warn方法和error方法也是在控制台输出信息，与log方法不同的是，warn方法输出信息时，会在前面加个黄色三角，表示警告；error方法输出信息时，在前边添加一个红色的叉号，表示错误信息。

console.dir(), console.dirxml()

dir方法用于对一个对象进行检查，并以易于阅读的格式显示。

console.assert()

console.assert方法主要用于程序运行过程中，进行条件判断，如果条件为 false，则将一个错误消息写入控制台。如果是 true，没有任何反应。

console.time(), console.timeEnd()

这两个方法用于计时，可以算出一个操作所花费的准确时间。

console.group(), console.groupEnd(), console.groupCollapsed()

console.group和console.groupEnd 用于显示信息分组。它只在输出大量信息时有用。
