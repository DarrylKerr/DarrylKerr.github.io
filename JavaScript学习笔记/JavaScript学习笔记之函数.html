JavaScript学习笔记 - 函数

发表于 2016-04-25 | 分类于 JavaScript学习笔记 |

怎么理解函数，简单来讲就是一段可以反复调用的代码块，可以接受不同的参数返回不同的值。

return 语句和递归

函数体内部的 return语句，表示返回。JavaScript 引擎遇到 return 语句，就直接返回 return 后面的那个表达式的值，后面即使还有语句，也不会得到执行。也就是说，return 语句所带的那个表达式，就是函数的返回值。return 语句不是必需的，如果没有的话，该函数就不返回任何值，或者说返回 undefined。

函数的作用域

定义

作用域（scope），指的是变量的存在范围，在 ES5 中，变量只有两种作用域：

  • 全局作用域，变量在整个程序中都存在，每个地方都能读取
  • 函数作用域，变量只在函数内部有效

参数

概述

函数在大多数情况下需要引入外部数据，不同的外部数据得到不同的结果，这种外部数据叫做参数。

参数的省略

函数的参数不是必需的，JS 允许省略参数

传递方式

函数参数如果是原始类型的值，传递方式是传值传递。那么，在函数内部修改参数的值，不会影响函数外部。

但是，如果函数参数是复合类型的值，传递方式是传址传递。也就是说，传入函数的是原始值的地址，因此在修改内部参数的时候，将会影响到原始值。看个例子：
