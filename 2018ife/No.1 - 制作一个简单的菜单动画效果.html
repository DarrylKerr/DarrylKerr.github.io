什么是 CSS 伪元素

CSS 的伪元素，之所以被称为伪元素，是因为他们不是真正的页面元素(不占用 DOM 节点)，html 没有对应的元素，但是其所有用法和表现行为与真正的页面元素一样，可以对其使用诸如页面元素一样的 css 样式，表面上看上去貌似是页面的某些元素来展现，实际上是 css 样式展现的行为，因此被称为伪元素。常见的伪元素比如:before :after等等。
